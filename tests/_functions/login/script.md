## Troubleshooting

- If stuck, may need manual CAPTCHA solving

### reCAPTCHA

- Usually shows image selection challenge
- Requires manual solving in test environment

### Already Logged In
- If URL contains "dashboard", function returns immediately
- No need to re-login
