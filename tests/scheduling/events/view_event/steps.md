# View Event Test

## Objective
Open and view the details of a scheduled group event instance.

## Prerequisites
- User is logged in
- A scheduled event exists (context: scheduled_event_id or scheduled_event_time)
- Browser is on Calendar page

## Steps
1. Verify on Calendar page
2. Find the scheduled event in the calendar (by time or context)
3. Click on the event
4. Verify event details are displayed (service name, date/time, attendees, etc.)

## Expected Result
- Event details dialog/page opens
- All event information is visible and correct
- Service name matches context
- Date/time matches context
