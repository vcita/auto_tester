# Edit Event Test

## Objective
Modify details of a scheduled group event instance (e.g., notes, capacity).

## Prerequisites
- User is logged in
- A scheduled event exists (context: scheduled_event_id)
- Browser is on event detail page or Calendar page

## Steps
1. Open the scheduled event
2. Click "Edit" button or option
3. Modify event details (e.g., add notes, change capacity if allowed)
4. Save changes
5. Verify changes are reflected in event details

## Expected Result
- Event details are updated successfully
- Changes are visible in the event view
- Event still appears in calendar at correct time
