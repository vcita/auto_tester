# Book a Consultation

## Objective
Book a consultation appointment as a new client

## Prerequisites
- Business has "Consultation" service enabled
- At least one time slot available in the next 7 days

## Steps
1. Go to the business booking page
2. Select the "Consultation" service
3. Pick the first available time slot
4. Fill in client details (name, email, phone)
5. Confirm the booking

## Expected Result
- Confirmation message is displayed
- Booking reference number is shown
- Summary of the appointment is visible
